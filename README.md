# kilimi84.github.io

## 새 글 쓰는 법

1. `_posts` 폴더에 새 파일 만들기
2. 파일 이름 형식: `YYYY-MM-DD-제목.md` (예: `2026-07-01-여행기.md`)
3. 파일 맨 위에 아래 형식 붙이기:

```
---
title: "글 제목"
date: 2026-07-01
tags: [태그1, 태그2]
---

여기부터 본문을 마크다운으로 쓰면 됩니다.
```

4. Commit 하면 1~2분 후 자동으로 사이트에 반영됩니다.

## 글 삭제/수정

`_posts` 폴더에서 해당 파일을 열어 수정하거나 삭제하면 됩니다.
